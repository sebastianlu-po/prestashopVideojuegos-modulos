# Entorno de Módulos - Tienda Videojuegos de PrestaShop

Este repositorio contiene funciones personalizadas para prestashop, importación de catálogo base, sincronización con otra tienda, y  avisos en la página principal. 

## Contenido del Repositorio

### 1. Módulos de PrestaShop
*   **Importación de catalogo**
*   **sincronización con otra tienda**
*   **alertas**

### 2. Código anterior a la creación de módulos
*   **login-php**: Desarrollo inicial de un sistema de autenticación realizado en PHP. Contiene una base de datos que añade usuarios al archivo y los saca del archivo para añadirlos a un array y poder usar la información de estos.
